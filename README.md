# TwiceParking
TwiceParking is VB .NET project that manage vehicle parking system. This project use QR code on card in order to identify user. After the transaction is completed, user will get  email that contain last transaction bill from the system.

Due to permission with the other team members who create this amazing project, only a several source code files will be post in this repository.
if you'd like to see the program, you can run the program with following methods.

*Methods*
1. import sistem_parkir.sql on mysql workbench
2. run the file on ProjectSistemParkir\bin\Debug\ProjectSistemParkir.exe
3. voila! just enjoy your piece of cake.
